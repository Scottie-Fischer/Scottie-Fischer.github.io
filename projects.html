Oscilloscope

An embedded project using a Cypress CY8CPROTO-063 MCU. The RTC is used as a trigger for the DMA to read the value from the ADC unit that takes a signal from GPIO pin. A buffer in RAM will be filled, once filled the average frequency will be calculated while a separate buffer is filled. The average of these frequencies will be output to the LCD every couple of seconds.

C GPIO ADC DMA LCD Display RTC Embedded

Ransomware Detection

This was a project in conjunction with Nutanix. The project is a randsomware detection application that monitors various signals from a cloud hypervisor and vSwitch to detect ransomware spreading. We used http status, number of digits, and url entropy to train a binary classification tree between benign and malicious traffic. The system also collects the Shannon's entopy of every file write, and will correlate with the classification of the ML model in a sliding window.
